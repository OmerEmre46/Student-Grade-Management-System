# Student Grade Management System

A simple yet efficient Student Grade Management System developed in C. This project utilizes a **Linked List** data structure to manage student records and provides file-based persistence, ensuring data is saved and loaded automatically.

## Features

* **Dynamic Data Management**: Uses Linked List to add and delete student records dynamically in memory.
* **File Persistence**: Automatically saves data to `ogrenciler.txt` and loads existing records when the program starts.
* **CRUD Operations**: Supports Create (Add), Read (Load), and Delete operations.
* **Error Handling**: Includes basic checks for file operations and memory management.
